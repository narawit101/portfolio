# 🚀 My Personal Portfolio

เว็บไซต์รวบรวมผลงาน (Portfolio) ที่พัฒนาด้วย Next.js 16 และ React 19 เน้นดีไซน์ที่ทันสมัย (Modern Dark UI) และการแสดงผลที่ลื่นไหล เพื่อนำเสนอทักษะและโปรเจกต์ต่างๆ ของผม

## ✨ ฟีเจอร์หลัก
- **Modern UI/UX**: ดีไซน์โทนเข้ม (Dark Theme) สวยงาม สบายตา
- **Responsive Design**: รองรับการใช้งานทุกหน้าจอ (Mobile, Tablet, Desktop)
- **Tech Stack Showcase**: แสดงทักษะความสามารถแยกตามหมวดหมู่ (Front-end, Back-end, Database, Tools)
- **Project Highlight**: ส่วนแสดงรายละเอียดผลงานพร้อมรูปภาพประกอบและ Tag เทคโนโลยีที่ใช้
- **Contact Section**: ช่องทางการติดต่อสื่อสารที่เข้าถึงง่าย

## 🛠️ เทคโนโลยีที่ใช้
### Front-End
- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://reactjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/), [DaisyUI](https://daisyui.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

### Back-End & Database (Featured in Projects)
- **Runtime**: Node.js, Express.js
- **ORM**: Sequelize
- **Database**: PostgreSQL, MongoDB, MySQL
- **Real-time**: Socket.IO

### Deployment & Tools
- **Deployment**: Vercel, Railway
- **DevOps**: Docker, Git
- **Storage**: Cloudinary

## 📂 โครงสร้างโฟลเดอร์
- `app/`: หน้าหลักและ Layout ของเว็บไซต์
- `components/`: UI Components แยกส่วนเพื่อให้ง่ายต่อการดูแล
- `data/`: ไฟล์ข้อมูลสำหรับ Projects และ Tech Stack
- `types/`: การกำหนด Type สำหรับ TypeScript
- `public/`: ไฟล์ Static เช่น รูปภาพและไอคอนต่างๆ

## 🚀 เริ่มต้นใช้งาน (Local Development)

1. ติดตั้ง Dependencies:
   ```bash
   npm install
   ```

2. รันโปรเจกต์ในโหมด Development:
   ```bash
   npm run dev
   ```

3. เปิดเบราว์เซอร์ไปที่ [http://localhost:3000](http://localhost:3000)
